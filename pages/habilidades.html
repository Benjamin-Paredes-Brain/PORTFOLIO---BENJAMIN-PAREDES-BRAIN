habilidades

HTML5

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear y estructurar el contenido de una página web.

CSS3

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para dar formato y diseño a la presentación visual de una página web.

Bootstrap

Bootstrap es un framework de diseño front-end que incluye una variedad de herramientas y componentes preconstruidos, como botones, formularios, tablas, y tipografía, para facilitar el desarrollo rápido de páginas web responsivas.

Git

Git es un sistema de control de versiones utilizado para administrar el código fuente de un proyecto de software y hacer seguimiento de los cambios en el mismo.

GitHub

GitHub es una plataforma web de alojamiento y colaboración para proyectos de software que utilizan Git como sistema de control de versiones. Permite a los desarrolladores compartir su código, realizar seguimiento de los cambios, y trabajar en equipo en un mismo proyecto.

Sass

SASS (Syntactically Awesome Style Sheets) es una extensión de CSS que ofrece funcionalidades adicionales, como variables, anidamiento de selectores, y funciones, para simplificar el proceso de creación de hojas de estilo.
